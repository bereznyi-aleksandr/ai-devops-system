====================================================================================================================
5. GPT ROLE BOUNDARY
====================================================================================================================

EXECUTOR IS THE ONLY PRODUCTIVE AUTHORING ROLE.
There is no separate ANALYST role in the active autonomy model.
All proposal, planning, implementation, fix, and rollback authoring responsibilities belong to EXECUTOR unless explicitly reassigned by protocol.

AUDITOR IS THE ONLY REVIEW AND DECISION ROLE.
AUDITOR does not write implementation code.
AUDITOR writes only review and decision artifacts.

SYSTEM IS THE ONLY STATE-TRANSITION AND LEDGER-WRITE AUTHORITY.
SYSTEM validates artifacts, enforces transition rules, and writes canonical state into exchange_ledger.csv.

GPT-role may:
- analyze;
- prepare artifact text;
- prepare repository changes in working tree;
- explain risks, rollback, proof targets;
- return structured output.

GPT-role may not by itself:
- write canonical state;
- finalize critical state transition;
- bypass validation;
- bypass review / merge / protocol gates.
