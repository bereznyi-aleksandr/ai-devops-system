---

## Claude → GPT/Codex (переключение на резервный)

Условия (любое из):
1. Claude Code не отвечает 2 цикла подряд
2. Claude Code Action временно недоступен
3. Основной контур завис более двух циклов куратора

Процесс:
1. Куратор обнаруживает блокер
2. Куратор самостоятельно принимает решение о переключении
3. Обновляет EXCHANGE.md: active_contour → gpt_codex
4. Добавляет событие в exchange.jsonl: CONTOUR_SWITCH
5. Запускает GPT_ANALYST
6. Сообщает оператору в Telegram: контур переключён

Подтверждение оператора НЕ требуется.

---

## GPT/Codex → Claude (возврат на основной)

Условия:
1. Claude Code снова доступен
2. Workflow проходит без ошибок

Процесс:
1. Куратор самостоятельно принимает решение о возврате
2. Обновляет EXCHANGE.md: active_contour → claude
3. Добавляет событие в exchange.jsonl: CONTOUR_SWITCH
4. Сообщает оператору в Telegram: контур восстановлен

Подтверждение оператора НЕ требуется.

---

## ЗАПРЕТЫ

1. Оба контура активны одновременно — ЗАПРЕЩЕНО
2. @analyst не запускает GPT_ANALYST
3. @codex ROLE=GPT_ANALYST не запускает Claude Analyst
