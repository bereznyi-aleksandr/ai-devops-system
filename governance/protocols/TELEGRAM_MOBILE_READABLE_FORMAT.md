# BEM-570 | Telegram Mobile Readable Format

Дата: 2026-05-17 | 21:26 (UTC+3)

## Решение

| Наименование | Описание | Обоснование |
|---|---|---|
| Telegram tables | Запрещены для оперативных уведомлений | На мобильном экране строки переносятся и становятся нечитаемыми |
| Формат | Короткие секции и отдельные строки | Удобно читать в Telegram |
| Канон | Шапка `Этап` и `Дорожная карта` сохраняется | Требование оператора |
| Табличный канон | Разрешён в repo reports, но не в Telegram notification body | В файлах таблицы читаемы, в Telegram — нет |

## Telegram notification format

```text
BEM-MAILBOX | AUDIT MAILBOX | workflow_runtime

Этап: 1/1 (100%)
Дорожная карта: 1/1 (100%)

Чек-лист:
✅ Новое сообщение mailbox обнаружено
✅ Отправитель определён
✅ Получатель определён

Сообщение:
От: Claude
Кому: GPT
Тема: ...
Файл: governance/audit_mailbox/...

Действие:
Прочитать файл и ответить в mailbox.
```
