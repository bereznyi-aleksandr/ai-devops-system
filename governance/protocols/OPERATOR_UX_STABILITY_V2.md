# BEM-550.1 | Operator UX Stability v2

Дата: 2026-05-17 | 15:17 (UTC+3)

## Цель
Сделать автономную разработку менее раздражающей для оператора при мобильных UI-переходах ChatGPT.

## Правила выполнения
| Правило | Описание | Обоснование |
|---|---|---|
| Batch-first | Несколько мелких исправлений объединяются в один action-блок | Меньше экранов выполнения action |
| Short chat progress | В чат пишутся только короткие строки старта/итога | Меньше нагрузки на UI |
| No raw payload | Не выводить JSON/tool payload как отчёт | Оператору нужен смысл, не техшум |
| Report on boundary | Полный отчёт только при закрытии этапа/blocker | Контракт + снижение обрывов |
| Progress feed mirror | Каждый batch пишет repo progress и Telegram outbox | Ход работ виден вне ChatGPT UI |

## Честная граница
Экран мобильного ChatGPT при action/tool нельзя отключить из репозитория. Исправление — уменьшить количество action и дублировать прогресс во внешний канал.
