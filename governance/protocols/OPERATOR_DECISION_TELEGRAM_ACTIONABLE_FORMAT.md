# BEM-574 | Operator Decision Telegram Actionable Format

Дата: 2026-05-17 | 21:41 (UTC+3)

## Правило
Telegram-сообщение для решения оператора должно быть самодостаточным. Оператор не обязан открывать repo-файл, чтобы понять, что именно согласовывать.

## Обязательные поля в Telegram

- Вопрос
- Варианты решения: 1 / 2 / 3
- Рекомендация аудиторов
- Как ответить
- Что будет после решения

## Запрещено
Нельзя отправлять оператору только путь к файлу без сути вопроса и вариантов ответа.
