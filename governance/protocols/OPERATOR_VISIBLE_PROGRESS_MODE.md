# BEM-548 | Operator Visible Progress Mode

Дата: 2026-05-17 | 15:08 (UTC+3)

## Причина
Оператор сообщил, что ChatGPT UI стал показывать неудобный экран/обрывы вместо нормального процесса разработки.

## Новое правило ответа
1. В чат выводить только короткий progress: `YYYY-MM-DD HH:MM (UTC+3) ▶/✅/❌ действие`.
2. Не выводить raw JSON/tool payload как основной ответ.
3. Не давать длинные промежуточные отчёты.
4. Полный канонический отчёт — только когда закрыт этап/блок или найден blocker.
5. Если ChatGPT UI уходит в экран профиля/отслеживания — продолжать repo execution и вернуть краткий статус после проверки.

## Граница ответственности
Агент не может исправить мобильный UI ChatGPT. Агент может снизить нагрузку на чат: короткие сообщения, меньше промежуточного текста, отчёт только по факту результата.
