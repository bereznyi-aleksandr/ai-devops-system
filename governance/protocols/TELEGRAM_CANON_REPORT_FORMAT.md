# BEM-560 | Telegram Canon Report Format

Дата: 2026-05-17 | 19:00 (UTC+3)

## Обязательная шапка Telegram-отчёта

```text
BEM-<N> | <НАЗВАНИЕ> | YYYY-MM-DD | HH:MM (UTC+3)

Этап: <X>/<Y> (<Z%>)
Дорожная карта: <X>/<Y> (<Z%>)
```

## Правило
В Telegram нельзя отправлять отчёт без количества задач этапа и процента по этапу.
