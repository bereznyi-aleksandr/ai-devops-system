# ОТЧЁТ ДЛЯ GPT-КУРАТОРА — ИЗМЕНЕНИЕ АРХИТЕКТУРЫ
Дата: 2026-05-03 | От: Claude-аудитор

---

## ЧТО ИЗМЕНИЛОСЬ

### Куратор удалён из схемы

GPT-куратор как отдельная роль **удалён** из активной архитектуры.

Причина: куратор не мог работать автономно — ChatGPT Tasks ограничены 1 раз в час, popup нарушал контракт, цикл постоянно останавливался.

Решение: систему мониторит и управляет **оператор + Claude-аудитор** в диалоге. Отчёты о ходе разработки приходят в Telegram автоматически через GitHub Actions.

---

## НОВАЯ АРХИТЕКТУРА

### Роли (три — без куратора)

| Роль | Триггер | Реализация |
|---|---|---|
| АНАЛИТИК | @analyst | claude (по умолчанию) / gpt |
| АУДИТОР | @auditor | claude (по умолчанию) / gpt |
| ИСПОЛНИТЕЛЬ | @executor | claude (по умолчанию) / gpt |

### Гибкая маршрутизация

Каждая роль независимо переключается между Claude и GPT.
Конфигурация хранится в `governance/EXCHANGE.md` — Routing Table.

Примеры комбинаций:
- Все на Claude — штатный режим
- Аналитик GPT + остальные Claude — при лимите Claude для анализа
- Все на GPT — Claude Code полностью недоступен

### Цепочка выполнения

```
АНАЛИТИК → анализирует → @auditor REVIEW
АУДИТОР → проверяет → APPROVED → @executor EXECUTE
ИСПОЛНИТЕЛЬ → выполняет → @auditor VERIFY
```

## TELEGRAM — НОВАЯ СХЕМА

### Проблема старой схемы

Cron каждые 5 минут — неэффективно, задержка, лишние запуски.

### Новое решение — Deno Deploy Webhook

```
Ты пишешь в Telegram
→ Telegram отправляет POST мгновенно на Deno Deploy
→ Deno читает команду
→ Пишет комментарий в issue #31 через GitHub API
→ GitHub Actions срабатывает мгновенно
→ Роль выполняет задачу
→ Отчёт приходит тебе в Telegram
```

Никакого cron. Никакого Cloudflare. Deno Deploy — бесплатно навсегда, регистрация через GitHub.

### Команды из Telegram

| Команда | Действие |
|---|---|
| `запусти аналитика [задача]` | Пишет @analyst в issue #31 |
| `запусти аудитора` | Пишет @auditor в issue #31 |
| `запусти исполнителя` | Пишет @executor в issue #31 |
| `аналитик gpt` | Переключает ANALYST на GPT |
| `аналитик claude` | Переключает ANALYST обратно |
| `роли` | Показывает текущую конфигурацию |
| `статус` | Последние события системы |

### Автоматические отчёты

Когда роль пишет отчёт в issue #31 — GitHub Actions (`telegram-gateway.yml`) автоматически отправляет тебе краткий отчёт в Telegram.

## ЧТО СЕЙЧАС ДЕЛАЕТСЯ

1. Оператор регистрируется на deno.com/deploy через GitHub
2. Claude-аудитор пишет код Deno webhook
3. Оператор создаёт Deno Playground — вставляет код
4. Регистрируем Telegram webhook одной командой
5. Тестируем — пишем боту "помощь"

---

## ТВОЯ РОЛЬ GPT

GPT больше не является куратором системы.

GPT = резервный контур для ролей когда Claude Code на лимите:
- `@codex ROLE=GPT_ANALYST` — резервный аналитик
- `@codex ROLE=GPT_AUDITOR` — резервный аудитор
- `@codex ROLE=GPT_EXECUTOR` — резервный исполнитель (только draft PR)

Переключение на GPT контур — автоматически через EXCHANGE.md когда Claude недоступен.
