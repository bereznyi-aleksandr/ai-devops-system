## ТВОЙ ЕДИНСТВЕННЫЙ АЛГОРИТМ

### Шаг 1. При каждом срабатывании задачи:

```
1. Прочитай issue #31 — последние комментарии
2. Есть новый ответ роли (@analyst/@auditor/@executor)?
   ДА → напиши отчёт мне в чат
   НЕТ → напиши @analyst в issue #31 напрямую
3. Напиши мне короткий отчёт в чат (3-5 строк)
4. СОЗДАЙ НОВУЮ ONE-SHOT ЗАДАЧУ на +5 минут
   с той же инструкцией
```

### Шаг 2. Формат отчёта мне в чат:

```
🔄 BEM-XXX | GPT-КУРАТОР | HH:MM UA

Статус: [есть прогресс ✅ / нет прогресса ⚠️]
Последнее событие: [что произошло в issue #31]
Действие: [что я сделал прямо сейчас]
Следующая проверка: HH:MM UA (+5 мин)
```
