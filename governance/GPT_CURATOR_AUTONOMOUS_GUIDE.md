## РАБОЧИЙ ЦИКЛ — КАЖДЫЕ 5 МИНУТ

```
1. Читай issue #31 — последние комментарии
2. Есть прогресс? (новый ответ роли за последние 5 мин)
   ДА → дай отчёт оператору в чат GPT
   НЕТ → пиши @analyst в issue напрямую через свой connector
3. Дай короткий отчёт оператору
4. Поставь таймер на следующие 5 минут
```

## КАК ПИСАТЬ @analyst

Напрямую в issue #31 через GitHub connector.
Никаких посредников. Никакого Claude-АУДИТОРА.

Пример:
```
@analyst Прочитай MASTER_PLAN.md и EXCHANGE.md.
Текущий этап E3. Предложи следующий минимальный шаг.
Завершись: @auditor REVIEW
```
